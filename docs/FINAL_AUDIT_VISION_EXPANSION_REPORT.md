# ONETOO Trust Ecosystem – Final Audit, Vision & Expansion Report
**Stav k:** 2026-01-25  
**Rozsah:** onetoo.eu · search.onetoo.eu · portal.onetoo.eu · hgpedu.eu

## Executive Summary (Final)
ONETOO je deterministický trust-hub prototyp („Trust Fabric“). Jadro je silné (trust root + search runtime), portál a škála zaostávajú.
Cieľ: maximalistický, globálne adaptovateľný ekosystém bez porušenia invariantov (no ML, audit-first, read-only UI).

## Cieľové schopnosti (maximalisticky)
- Registry pre typy: publisher, organization, agent, model, dataset, software, research_project, standard, policy
- Každá entita: stable id, status/lane, score, decision-trace, proofs, history
- Search (Phase 4+): v2 structured + facets + explain + entity endpoints + federation
- Portal: registry browser, entity detail, proofs, transparency, onboarding guide (read-only)

## Roadmap (skrátene)
- Fáza I: portál = registry browser + HGPEdu flagship + canonical URL cleanup + transparency timeline
- Fáza II: adoption kit + validators + SDK (JS/Python) + examples
- Fáza III: federácia trust rootov + cross-registry search + external audits
