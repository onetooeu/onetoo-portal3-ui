Explain (Mozart / Maximalist)

Deterministic, auditable scoring + trust signals, designed to remain verifiable for a decade. This page explains what the portal shows and why.

  • Read-only by default — it visualizes published artifacts.
  • Deterministic — same input, same output (no hidden ML decisions).
  • Composable — UI, worker API, and trust-root artifacts can evolve independently.
